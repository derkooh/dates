# dates

A date table for date/datetime wrangling.


| Column    | Description        |
| ---       | ---                |
| date      | Date in yyyy-mm-dd |
| month     | Month number       |
| yearmonth | yyyy-MMM format    |
| monthname | MMM format         |
| day       | dd format          |
| yyyymmdd  | mmddyyyy format    |
| year      | yyyy               |
| fod       | first time of day  |
| eod       | last time of day   |
| fow       | first of week      |
| eow       | end of week        |
| fom       | first of month     |
| eom       | end of month       |
| foq       | first of quarter   |
| eoq       | end of quarter     |
| foy       | first of year      | 
| eoy       | end of year        |
| dow       | Day of week        |
| week      | week number        |
| yearq     | yyyy-quarter       |
| fy        | fiscal year        |
| q         | quarter            |
| mm        | mm format          |
| dd        | dd format          |
